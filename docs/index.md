# Todo App - Three Horizons Golden Path

Full-stack Todo application built with the Three Horizons Agentic DevOps Platform.

## Stack

| Layer | Technology |
|-------|------------|
| Frontend | React 18 + TypeScript + Vite + Tailwind CSS |
| Backend | Node.js + Express + Prisma + Redis + Winston |
| Infrastructure | Terraform + Azure |
| Testing | Playwright E2E + Jest Unit |
| CI/CD | GitHub Actions |
| AI | 8 GitHub Copilot Agents |
